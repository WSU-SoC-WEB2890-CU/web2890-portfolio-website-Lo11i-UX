Lauren Photography

A mobile-first photography portfolio designed to highlight portrait and lifestyle work while keeping navigation simple and accessible.

Role: UX/UI Designer Tools: Figma, HTML, CSS Course: WEB 1400

Overview

This project was created to showcase my work in a clean, modern layout that looks great on mobile devices. The design uses large imagery, clear typography, and simple navigation.

Process

I started with mood boards and sketches, then moved into low-fidelity wireframes in Figma. After feedback from classmates, I created high-fidelity mockups and coded the layout using mobile-first CSS.

Key Decisions

  • Used a light blue color palette to keep the site airy and friendly.
  • Prioritized large tappable buttons and clear labels for mobile users.
  • Ensured sufficient color contrast and keyboard-friendly focus styles for accessibility.

What I Learned

This project helped me practice responsive image grids, spacing, and typography. I also strengthened my understanding of how to support photographers with layouts that showcase their work first.
