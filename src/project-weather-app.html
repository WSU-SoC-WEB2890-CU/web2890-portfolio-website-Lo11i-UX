Weather Application

A JavaScript-powered weather app that fetches real-time data from WeatherAPI and remembers the user's last searched city and preferred temperature unit.

Role: Front-End Developer Tools: HTML, CSS, JavaScript, WeatherAPI Concepts: API calls, async/await, localStorage

Project Overview

This Weather Application lets users type in a city, choose Celsius or Fahrenheit, and instantly see the current conditions, temperature, and humidity.
