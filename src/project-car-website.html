Cardeal One-Page Marketing Site

A responsive, one-page marketing website for a fictional car-buying company that gives users an instant cash offer for their car.

Role: UX Designer & Front-End Developer Tools: Bootstrap 5, SASS, HTML, Git Course: WEB 2890 - Project 1

Project Overview

For this assignment, I acted as the front-end designer and developer for Cardeal, a fictional company that gives users cash for their car. My task was to build a customized one-page marketing site using Bootstrap and SASS.

The main goals were to:

  • Showcase Cardeal's process and advantages clearly.
  • Make getting an offer feel fast, safe, and trustworthy.
  • Customize Bootstrap so the page looked branded, not generic.

My Responsibilities

  • Created a color palette, typography scale, and layout system using SASS variables and partials.
  • Coded a fully responsive one-page layout with Bootstrap 5, customizing components with SASS.
  • Implemented the main interaction patterns: hero valuation form, testimonial carousel, FAQ accordion, and recent-purchase cards.
  • Focused on accessibility, spacing, and clear visual hierarchy for all device sizes.

Key Features

Advantages of Cardeal

To differentiate Cardeal from traditional dealerships, I designed an "Advantages" section that highlights key benefits: competitive offers, willingness to buy imperfect cars, no obligation to buy a car from them, protection from scammers, nationwide locations, and theirr charitable donations. Instead of a plain bullet list, I used cards and icons to visually group each advantage.

Testimonial Carousel

The site features a carousel with five customer testimonials, each showing a quote, name, location, star rating, and date.I customized the Bootstrap carousel with new controls, card-style slides, and adjusted spacing so it felt unique and on-brand rather than like the default Bootstrap component.

Recent Purchases cards

At the bottom of the page, I used Bootstrap cards to showcase three recently purchased vehicles (Toyota 4-Runner, Ford Fusion, and Honda Civic). Each card lists year, mileage, price, full style, condition rating, fuel economy, and a short description with a "More Details" button.

Design & Development Approach

  • Color & branding: Customized the SVG logo (changing the car color) and built a SASS color system so the hero, buttons, and component accents all feel unified.
  • SASS partials: Broke styles into partial files for variables, typography, components, and layout, then compiled them into a single CSS file.

What I Learned

This project helped me get comfortable using Bootstrap 5 and SASS together, while still making the design feel custom and brand-driven. I practiced planning a one-page marketing flow, balancing business goals (get offers) with user needs (clarity, safety, and trust), and paying attention to details like equal card heights and mobile behavior.

Project Details

  • Class assignment for WEB 2890.
  • Built as a one-page site with Bootstrap and SASS.
  • All text and photos were provided by instructor; layout and styling were my own.
